# Vue2-Starter

> vue2.0 全家桶启动模板

```
- Vue2.0
- Vue-router2.0
- Vuex2.0
- axios
- element-ui
```
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
# 启动网站
```
cd server && npm install
cd ../ && npm run serve
```
